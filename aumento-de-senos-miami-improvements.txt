------------------------------------------------------------------------------
SECTION 1 — COMPLETED: META TAG IMPROVEMENTS
------------------------------------------------------------------------------

All changes were applied to: articlemetas.cshtml

1. ADDED — og:site_name
   Explicitly declares the site name for social sharing previews.
   <meta property="og:site_name" content="Avana Plastic Surgery" />

2. ADDED — og:locale (es_US)
   Signals that the content is in Spanish targeting the US market.
   Previously missing — social crawlers were inferring language incorrectly.
   <meta property="og:locale" content="es_US" />

3. ADDED — og:locale:alternate (en_US)
   Signals that an English version of this page exists, helping Facebook
   and other platforms link both language versions.
   <meta property="og:locale:alternate" content="en_US" />

4. FIXED — og:url (relative → absolute URL)
   Open Graph URLs must be absolute. The previous value was relative,
   which could cause duplicate content issues in social crawlers.
   <meta property="og:url" content="https://avanaplasticsurgery.com/espanol/aumento-de-senos-miami" />

5. ADDED — article:tag
   Adds a content tag for the article Open Graph type, improving
   categorization in social platforms.
   <meta property="article:tag" content="Aumento de Senos" />

6. FIXED — og:image (relative → absolute URL)
   Same issue as og:url — image URL must be absolute for correct
   rendering in WhatsApp, Facebook, iMessage, etc.
   <meta property="og:image" content="https://avanaplasticsurgery.com/themes/avana/images/services/910x960/breast-augmentation.jpg" />

7. ADDED — og:image:alt, og:image:width, og:image:height
   Image alt text improves accessibility audits and AI-citation signals.
   Width/height allow social platforms to pre-size the preview box
   without having to download the image first.
   <meta property="og:image:alt"    content="Woman showing breast augmentation at Avana Plastic Surgery Miami" />
   <meta property="og:image:width"  content="910" />
   <meta property="og:image:height" content="960" />

8. FIXED — canonical (relative → absolute URL)
   Canonical tags must use absolute URLs. Relative canonicals are
   technically invalid per Google's guidelines.
   <link rel="canonical" href="https://avanaplasticsurgery.com/espanol/aumento-de-senos-miami" />
